# Legacy egui prototype

The original Nust editor: pure Rust, eframe/egui, four resizable panes, command palette.
Kept here as a reference for UX decisions — do **not** import code from it into the Tauri rewrite.

## Run

```powershell
cd legacy
cargo run
```

Uses stable Rust, no nightly required. See git history before May 2026 for the full evolution.
